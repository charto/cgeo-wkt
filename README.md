cgeo-wkt
========

Adds WKT export support to [cgeo](https://github.com/charto/cgeo).

Usage
-----

```TypeScript
import * as cgeo from 'cgeo';
import 'cgeo-wkt';

const point = new cgeo.Point(12, 34);

console.log(point.toWKT());
```
